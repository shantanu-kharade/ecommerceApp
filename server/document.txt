API Endpoints :
User Registration: POST /api/auth/register
User Login: POST /api/auth/login
Get All Products: GET /api/products
Get Product Details: GET /api/products/:productId
Add Product to Cart: POST /api/cart/add
Remove Product from Cart: POST /api/cart/remove
Checkout: POST /api/checkout
Get User Profile: GET /api/user/profile
Update User Profile: PUT /api/user/profile
Get Order History: GET /api/user/order
Admin - Add New Product: POST /api/admin/products
Admin - Update Product: PUT /api/admin/products/:productId
Admin - Delete Product: DELETE /api/admin/products/:productId
